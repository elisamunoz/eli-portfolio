#### Default:

This component uses [fontawesome V4.7](https://fontawesome.com/v4.7.0/icons/), please check available icon names.
```jsx
<Icon />
```


#### With custom icon-name:

```jsx
<>
  <Icon name='times' />
  <Icon name='user' />
  <Icon name='check' />
</>
```


#### Clickable:

```jsx
<Icon name='times' onClick={()=> alert('Icon clicked')} />
```
